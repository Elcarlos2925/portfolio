The Drop Coffee.

(Window Forms)

Este proyecto consiste en el desarrollo de un programa ejecutable para Windows, diseñado como un punto de venta que permite gestionar las operaciones de una cafetería ficticia llamada The Drop Coffee.

La aplicación ofrece un menú principal sencillo donde se visualizan todos los productos disponibles, Incluye botones de información individual por producto, útiles en caso de que el cliente solicite detalles específicos de sus alimentos, además, incorpora algoritmos para calcular automáticamente el desglose del total de cada pedido, una simulación de ticket que almacena cada venta realizada y módulos de administración de usuarios para un mejor control del sistema.

Tecnologías Utilizadas

C#

.Net Framework

MySQL
